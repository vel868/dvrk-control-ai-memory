# Public repo security rules

Этот GitHub-репозиторий публичный. Его можно использовать только как безопасную AI-память проекта.

## Нельзя добавлять

- `.env`
- пароли
- YooKassa Secret Key
- Tilda Webhook Token
- WB API Token
- логин/пароль Basic Auth
- `/etc/nginx/.htpasswd_dvrk`
- дампы баз `.sql`, `.sql.gz`
- backup-архивы `.tar.gz`
- `/etc/letsencrypt` и SSL private keys
- персональные данные покупателей: телефоны, email, адреса, ФИО в виде списков заказов
- полные логи с персональными данными

## Можно добавлять

- архитектуру проекта
- бизнес-логику
- названия баз и таблиц
- безопасные команды без секретов
- список серверов и доменов
- правила разработки
- changelog
- открытые задачи
